  1. Install Git
  2. Reminder about newlines, i.e. git config --global auto.crlf input, and install EditorConfig
  3. Install Java
  4. Ensure JAVA_HOME and PATH configured properly
  5. Install IDE (IntelliJ)
  6. Set up GitHub account
  7. Set up Heroku / Fly.io account
